# 🚀 Swift Fundamentals

This repository contains my structured journey of learning the Swift programming language.

## 📚 Structure

Each folder represents a core concept of Swift:

- 01-basics — variables, data types, control flow  
- 02-optionals — optional values, unwrapping  
- 03-collections — arrays, dictionaries, sets  
- 04-functions-closures — functions and closures  
- 05-structs-vs-classes — value vs reference types  
- 06-enums — enums and associated values  
- 07-protocols — abstraction and contracts  
- 08-error-handling — try/catch, error flow  
- 09-generics — reusable and flexible code  
- 10-concurrency — async/await, threading  

---

## 🧠 Learning Approach

For each topic:
- Study theory
- Write examples
- Solve a small task
- Document understanding

## 🎯 Goal
